§ Playback

enum AnimaScript.Anima.Playback
strong

Playback mode.

Enumerator
HALT 

Halt playback.

When used in Play() it will be the same as Stop().

ONCE 

Play once.

Playback is stopped once it hit the outside of animation.

LOOP 

Loop play.

Loop infinitely, to stop call Stop().

PINGPONG 

Ping-pong play.

Loop infinitely, invert part speed in each border, to stop call Stop().
